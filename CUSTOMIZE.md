# Customize Checklist

## 第一批必须替换的占位内容

### about.html
- `待补：公众号名称 / 文章链接 / 二维码页`
- `待补：https://youtube.com/@your-channel`
- `待补：hello@example.com`

### 全站署名
- 页脚当前为 `LNOTM`
- 如需改为个人名、品牌名或双语署名，可统一替换 HTML 页脚和 `README.md`

### 站点链接
如果未来不是部署到 `LNOTM/my-blog`：
- 修改 `feed.xml`
- 修改 `sitemap.xml`
- 修改 `robots.txt`
- 修改 `site.webmanifest` 的 `start_url`

## 第二批可选优化

- 替换 `assets/icon.svg` 为你的正式站点图标
- 增加公众号二维码图片与真实外链
- 给 `projects.html` 中的项目卡片替换为真实项目
- 把示例文章换成你的正式首批内容
