# Publishing Checklist

## Before Publishing
- Confirm the title is specific and searchable.
- Add a one-paragraph summary.
- Check internal links and related article links.
- Add one distribution sentence for WeChat or YouTube.

## Before Commit
- Save screenshots if the post changes layout.
- Update archive.html if needed.
- Update feed.xml if the post is public.
- Update sitemap.xml if a new page was added.

## After Publishing
- Copy the article link to your public channels.
- Record what changed in the repository commit message.
- Note the next article or update idea while context is fresh.
