WHY THIS VERSION

这版更适合上线，不是因为“更花”，而是因为结构终于站稳了。

  • 文章是真正的独立页面，不再依赖浮层内容。
  • 保留像素风，但正文排版、间距和信息层级更适合长期阅读。
  • 场景化视觉被安放到合适位置，不压过内容本身。
  • 目录结构天然适合 GitHub Pages，也方便以后切到静态站生成器。
CONTENT SYSTEM

从“页面特效”转向“长期写作系统”

首页负责筛选与导览，文章页负责沉浸阅读，项目页负责展示持续产出，模板页负责说明设计结构，关于页负责集中联系方式。每一页都有清楚职责。

NEXT STEP

后续最值得接入的是评论、分发和真实链接体系

例如 Giscus、RSS、公众号落地页、YouTube 频道和真实邮箱。这样博客会从“能看”进一步变成“能维护、能传播”。
